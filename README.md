# 簡易CAD — 免費線上 2D 繪圖

**[▶ 點這裡直接使用](https://sancola1219-collab.github.io/simple-cad/)**(瀏覽器直接開,免安裝、免註冊)

給買不起商用 CAD 的人的免費 2D 繪圖軟體。單一 HTML 檔、零相依、完全離線可用 —
也可以下載 `index.html` 存到電腦,雙擊即用,永久免費。

## 功能

- **繪製**:線段、矩形、圓、圓弧、折線、文字、尺寸標註、量測(快捷鍵 L/R/C/A/P/T/D/M)
- **手機可用**:觸控繪圖、雙指縮放、控制點拖曳、選取快速操作列;**可安裝成 App、離線使用**
- **可直接開啟 DWG 與 DXF 檔**(內建 GNU LibreDWG 引擎,含圖塊展開與中文編碼;DWG 需線上使用)
- **精準**:格點鎖點、物件鎖點(端點/中點/圓心/四分點)、正交(Shift)、繪製中直接輸入長度/半徑
- **編輯**:框選(左包右碰)、移動、複製 Ctrl+D、復原/重做、圖層(顏色/隱藏/鎖定)
- **進階編輯**:旋轉、鏡射、偏移(平行複製)、修剪、延伸、陣列;半徑/角度標註;物件屬性數值編輯;設備符號圖庫
- **匯出 PDF**:A4/A3、比例 1:1~1:200 或符合頁面、標題欄 — 向量輸出,印出來用尺量就是準的
- **匯出 DXF**(可給 AutoCAD/ezdxf)、SVG、PNG;專案存成 .json 檔
- **效能**:實測 40 萬圖元的工廠平面圖可流暢操作(場景快取+空間索引)
- 單位 mm、Y 向上、原點左下;介面繁體中文

## 隱私

所有圖面資料只存在你自己的瀏覽器(localStorage)與你下載的檔案裡,**不會上傳到任何伺服器**。

## 匯出 PDF 的方法

按「匯出 PDF」→ 選紙張與比例 →「列印/存成 PDF」→ 印表機選「另存為 PDF」。
